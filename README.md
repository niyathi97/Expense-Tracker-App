# 💰 Expense Tracker App using Data Science

---

## 📌 Project Overview

The **Expense Tracker App** is a data-driven application designed to help users track, analyze, and visualize their spending habits.
It converts raw transaction data into meaningful insights using data analysis and visualization techniques.

---

## 🎯 Problem Statement

Managing personal expenses manually is difficult and unorganized.
Users often struggle to:

* Track where money is spent
* Identify unnecessary expenses
* Plan budgets effectively

---

## 💡 Solution

This project provides:

* Expense tracking system
* Category-wise analysis
* Monthly trend visualization
* Interactive dashboard using Streamlit

---

## 🚀 Features

* 📊 Category-wise spending analysis
* 📈 Monthly expense trends
* 📅 Weekday spending insights
* 🔍 Interactive filters
* 💻 Streamlit web dashboard
* 📁 Clean dataset export

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/Expense-Tracker-App.git
cd Expense-Tracker-App
```

### 2. Create virtual environment

```
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

---

## ▶️ How to Run

### Run analysis script

```
python main.py
```

### Run Streamlit app

```
streamlit run app.py
```
